# Student Performance Exploratory Data Analysis (EDA)

## Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Student Performance dataset.
The goal of this analysis is to understand how different factors such as study time, parental education, lifestyle habits, and demographic characteristics influence student academic performance.

Through visualization and statistical summaries, the project identifies patterns and relationships within the dataset.

---

## Dataset Source

The dataset used in this project was obtained from **Kaggle**.

It contains information about students' academic performance along with various demographic, social, and study-related factors.

---

## Dataset Description

The dataset includes the following types of information:

* Demographic information (age, gender, address)
* Family background (parent education, family size, parent status)
* Study habits and lifestyle (study time, free time, going out, alcohol consumption)
* Academic performance (G1, G2, G3 grades)

The final grade **G3** represents the student's final academic performance.

---

## Tools and Libraries Used

The analysis was performed using the following Python libraries:

* **Python**
* **Pandas** – Data manipulation and analysis
* **NumPy** – Numerical computations
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical visualization

---

## Analysis Performed

### 1. Data Understanding

* Dataset loading and inspection
* Checking data types
* Identifying numerical and categorical variables

### 2. Univariate Analysis

Analysis of individual variables to understand their distribution.

Techniques used:

* Histograms
* Boxplots
* Countplots
* Summary statistics

### 3. Bivariate Analysis

Analysis of relationships between two variables.

Techniques used:

* Scatter plots
* Box plots
* Count plots with hue
* Correlation analysis

---

## Key Insights

Some important observations from the analysis:

* Students who spend more time studying generally achieve better grades.
* Students with more past failures tend to have lower final grades.
* Early academic performance (G1 and G2) strongly influences the final grade (G3).
* Absences may negatively impact student performance.
* Gender does not show a significant difference in final grades.

---

## Conclusion

This exploratory data analysis provides insights into how various academic, demographic, and lifestyle factors relate to student performance. The findings highlight that study habits and previous academic results play an important role in determining final grades.
